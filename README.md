# HueFit Closet

A front-end prototype for an outfit recommendation app where users can photograph the clothing and accessories they already own, apply filters, and generate outfit possibilities that match the occasion.

## Run

Open [index.html](/Users/mercedes/Documents/New project/index.html) in a browser.

## What This Prototype Includes

- A mobile-friendly layout designed to work cleanly on iPhone-sized screens
- Camera-friendly clothing upload using a photo input with `capture="environment"`
- A wardrobe form that stores category, color, style, season, dress level, and texture
- Outfit generation filters for occasion, mood, weather, dress code, color palette, and priority piece
- A stronger recommendation template that explains why each suggested outfit works
- Appearance controls for theme family, card shape, quiet mode, and focus mode
- Local browser storage for both wardrobe items and UI preferences

## Notes

- Outfit recommendations are generated in the browser from simple matching rules
- Uploaded images and settings are stored in local storage on the current browser
- No backend or AI image analysis is included yet in this prototype
